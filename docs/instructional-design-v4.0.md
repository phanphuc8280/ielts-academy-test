# Instructional Design v4.0

The course is designed as an IELTS Task 2 tutor, not a content archive.

## Lesson law

One lesson = one skill. Practice and Homework must practise only that skill.

## Exercise law

An exercise is not a prompt. A prompt is context. Every exercise must include Objective, Situation, Task, Requirement, Example, Checklist and Expected Outcome.

## Practice vs Homework

Skill Practice is guided and scaffolded. Homework is independent and uses a new context.

## Mastery law

Mastered requires feedback, rewrite and reflection.

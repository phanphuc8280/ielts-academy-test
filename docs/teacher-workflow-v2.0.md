# Teacher Workflow v2.0

Teacher Mode is designed for checking the student's IELTS Writing learning loop:

1. Check Today's Plan and Review Queue.
2. Open homework waiting for review.
3. Add Teacher Notes: strengths, weaknesses, next homework, teacher comment.
4. Mark lesson as Reviewed.
5. Require Rewrite before Mastered.
6. Save band scores in Essay Workspace to update Band Tracker.
7. Use Custom Assignment when the student runs out of suitable practice.

Do not mark a lesson Mastered unless the student has practice, homework, feedback, and rewrite/reflection.

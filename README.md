# IELTS Academy v7.0 - 100MB Premium Learner LMS

Bản v7.0 tập trung vào trải nghiệm học viên, style hiện đại và kho học liệu lớn hơn.

## Nâng cấp chính

- Style v7 toàn diện: premium glass, neon gradient, light theme rõ hơn, hover lift, focus border, smooth tabs.
- Thêm Resource Library: vocabulary decks, idea capsules, skill packs, PDF packs, visual learning assets.
- Nâng cấp lesson PDFs vào `resources/lesson-pdfs-v7/`.
- Thêm premium PDFs trong `resources/v7-premium-pdfs/`.
- Thêm wallapers/textures trong `assets/images/v7-wallpapers/` và `assets/images/v7-textures/` để package khoảng 100MB.
- Giữ Ask Chị Context Engine, login, localStorage, multi-user, teacher/admin dashboard từ v6.

## Cách mở offline

Mở `index.html` bằng Chrome/Edge.

## Deploy GitHub Pages

Copy nội dung bên trong `ielts-academy-v7.0` vào root của repo, giữ `.git` và `.nojekyll`, commit rồi push.

## Lưu ý

Bản v7.0 cố tình nặng theo yêu cầu để có nhiều assets/PDF học liệu. Không có file đơn lẻ vượt 100MB.

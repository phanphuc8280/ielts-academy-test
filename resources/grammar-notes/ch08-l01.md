# Grammar Notes - 8.1 Task Response: trả lời đủ và sâu hơn

## Grammar focus

For this lesson, grammar correction is secondary unless the current skill is grammar. The priority is **Task Response Depth**.

## Useful sentence patterns

- One reason is that ...
- This can lead to ...
- For example, ...
- Although this may be true in some cases, ...
- As a result, ...

Do not ask AI to correct the whole essay if the lesson only practises one skill.

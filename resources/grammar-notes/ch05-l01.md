# Grammar Notes - 5.1 Topic Sentence: giám khảo hiểu ngay luận điểm

## Grammar focus

For this lesson, grammar correction is secondary unless the current skill is grammar. The priority is **Topic Sentence**.

## Useful sentence patterns

- One reason is that ...
- This can lead to ...
- For example, ...
- Although this may be true in some cases, ...
- As a result, ...

## Warning

Do not ask AI to correct the whole essay if the lesson only practises one skill.

# Grammar Notes - 5.4 Benefit / Drawback: phát triển mặt lợi và hại

## Grammar focus

For this lesson, grammar correction is secondary unless the current skill is grammar. The priority is **Benefit / Drawback Explanation**.

## Useful sentence patterns

- One reason is that ...
- This can lead to ...
- For example, ...
- Although this may be true in some cases, ...
- As a result, ...

## Warning

Do not ask AI to correct the whole essay if the lesson only practises one skill.

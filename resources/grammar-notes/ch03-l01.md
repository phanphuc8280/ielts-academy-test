# Grammar Notes - 3.1 Main Ideas: tìm 2 ý chính rõ ràng

## Grammar focus

For this lesson, grammar correction is secondary unless the current skill is grammar. The priority is **Main Ideas**.

## Useful sentence patterns

- One reason is that ...
- This can lead to ...
- For example, ...
- Although this may be true in some cases, ...
- As a result, ...

Do not ask AI to correct the whole essay if the lesson only practises one skill.

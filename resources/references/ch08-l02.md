# Reference - 8.2 Coherence: progression giữa các câu

## Lesson pipeline

Overview → Learning Objective → Theory → Examples → Guided Analysis → Skill Practice → Homework → AI Feedback → Rewrite → Reflection → Mastered

## Mastery rule

A lesson is Mastered only after practice, homework, feedback, rewrite and reflection. Reading theory is not enough.

## Ask Chị focus

Please focus on **Coherence Progression** only. Do not review unrelated skills unless the student explicitly asks.

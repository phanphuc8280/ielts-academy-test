# Vocabulary Sheet - 5.3 Cause–Effect: giải thích nguyên nhân và kết quả

Skill focus: **Cause–Effect Explanation**

## Useful phrases

- clear position
- relevant main idea
- practical consequence
- long-term impact
- social responsibility
- directly related to the question

## Collocation practice

1. develop a clear argument
2. provide a realistic example
3. address the exact question
4. avoid vague language
5. improve paragraph unity

## IELTS example sentence

A clear argument should be specific enough to answer the question, but not so narrow that it cannot be developed.
